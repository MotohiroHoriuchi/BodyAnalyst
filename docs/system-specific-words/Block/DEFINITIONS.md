# Block

## Definition
A **Block** is a modular, independent unit within the Dashboard's grid layout. It encapsulates a specific data visualization, functional widget, or summary card. Blocks are the fundamental atomic units that users can add, remove, resize, and rearrange to customize their Dashboard view.

Each Block is reconstructed dynamically based on persisted metadata, which defines its data source, visualization type, and layout properties.

## Japanese Explanation
**Block (ブロック)**
ダッシュボードのグリッドレイアウトを構成する、独立したモジュール単位です。
グラフ、機能ウィジェット、サマリーカードなど、特定のデータの可視化や機能を内包します。
ユーザーはこれらのブロックを追加、削除、リサイズ、並べ替えすることで、ダッシュボードを自由にカスタマイズできます。

各ブロックは永続化されたメタデータ（データソース、表示タイプ、レイアウト情報など）に基づいて、動的に再構築されます。

## Related Terms
- **Dashboard**: The container screen that manages the layout of Blocks.
- **[Visualization Engine](../Visualization_Engine/DEFINITIONS.md)**: The component responsible for rendering the content within a data-centric Block.
