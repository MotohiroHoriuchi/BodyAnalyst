# Screen Definition: Body Composition Record

## Overview
This screen is dedicated to tracking physical body metrics over time, such as weight and body fat percentage.

## Key Features
- **Metric Entry**: Input for weight, body fat %, and optionally muscle mass or other measurements.
- **Trend Visualization**: Charts or graphs showing changes in body composition over weeks, months, or years.
- **Log History**: A list of past entries with the ability to edit or delete them.

## Data Requirements
- Weight and body composition records
- Date/time of measurement
