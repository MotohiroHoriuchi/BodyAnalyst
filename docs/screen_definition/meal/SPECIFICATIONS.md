# Screen Definition: Meal Record

## Overview
The Meal Record screen is used to track nutritional intake. Users can log individual food items and see their total daily consumption.

## Key Features
- **Food Search**: Search from a database of food items.
- **Meal Entry**: Categorize entries into Breakfast, Lunch, Dinner, or Snacks.
- **Nutritional Breakdown**: Real-time calculation of total Calories, Protein, Fat, and Carbohydrates (PFC).
- **Custom Food Creation**: Allow users to add their own food items if not in the master list.

## Data Requirements
- Food master data
- Daily meal records
- User-specific custom food data
