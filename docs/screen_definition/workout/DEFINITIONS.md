# Screen Definition: Workout Record

## Overview
This screen allows users to log their strength training and other physical exercises. It focuses on tracking sets, repetitions, and weight for various exercises.

## Key Features
- **Exercise Search/Selection**: Choose from a master list of exercises.
- **Session Logging**: Track multiple exercises within a single workout session.
- **Set Tracking**: Record weight and reps for each set.
- **Rest Timer**: A built-in timer to manage rest periods between sets.
- **History View**: Access to previous records for the selected exercise to track progress (e.g., 1RM trends).

## Data Requirements
- Exercise master data
- Workout session records
- Historical performance data for specific exercises
