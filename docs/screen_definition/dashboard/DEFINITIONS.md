# Screen Definition: Dashboard (TOP)

## Overview
The Dashboard is the home screen of the application, providing a high-level summary of the user's progress and quick access to logging features.

## Key Features
- **Daily Progress Summary**: Visualization of today's calorie and macro intake against goals.
- **Recent Activities**: A quick view of the last workout and weight entry.
- **Quick Actions**: Buttons to quickly navigate to Workout, Meal, or Weight logging.

## Data Requirements
- Daily goals (from User settings)
- Today's meal records summary
- Latest weight record
- Latest workout session data
