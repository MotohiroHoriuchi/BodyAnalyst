# Body Analyst ユーザーガイド

## アプリ概要

Body Analystは、ボディメイク（筋トレ・ダイエット）を効率的に管理するためのPWA（Progressive Web App）です。体重・食事・トレーニングを記録し、データを可視化することで、科学的根拠に基づいた体づくりをサポートします。

## 主な機能

### 1. ホーム画面（ダッシュボード）

アプリを開くと最初に表示される画面です。

- **今日のカロリー進捗**: 目標カロリーに対する達成度を円グラフで表示
- **PFCバランス**: タンパク質(P)・脂質(F)・炭水化物(C)の摂取状況
- **体重トレンド**: 直近の体重推移をグラフで表示
- **週間サマリー**: 1週間の平均カロリー・体重の変化

### 2. 食事記録

毎日の食事を記録する画面です。

- **日付選択**: カレンダーから記録したい日を選択
- **食事タイプ**: 朝食・昼食・夕食・間食を選択
- **食材検索**: 170種類以上の食材マスタから選択
  - 肉類、魚介類、卵・乳製品、豆類、穀類、野菜、果物など
  - プロテインやサプリメントも登録済み
- **カスタム食材**: マスタにない食材は自分で追加可能
- **分量入力**: グラム単位で入力し、栄養素を自動計算

### 3. トレーニング記録

ワークアウトを記録する画面です。

- **ワークアウト開始/終了**: セッションの開始・終了時間を記録
- **種目選択**: 28種類のトレーニング種目から選択
  - 胸: ベンチプレス、ダンベルフライなど
  - 背中: デッドリフト、懸垂など
  - 肩: オーバーヘッドプレス、サイドレイズなど
  - 腕: バーベルカール、トライセプスプッシュダウンなど
  - 脚: スクワット、レッグプレスなど
  - 体幹: プランク、クランチなど
- **セット記録**: 重量×回数を入力
- **レストタイマー**: セット間の休憩時間をカウント
- **1RM推定**: 入力した重量・回数から推定1RM（1回だけ挙げられる最大重量）を表示
- **過去のワークアウト**: 履歴を確認・編集・削除

### 4. 分析画面

記録したデータをグラフで可視化する画面です。

- **カスタマイズ可能なウィンドウ**: 見たいデータを自由に配置
- **表示可能なデータ**:
  - 体重・体脂肪率
  - カロリー・PFC（タンパク質・脂質・炭水化物）
  - 総トレーニングボリューム
  - 種目別ボリューム・1RM・最大重量
- **期間調整**: 1週間〜1年の範囲で表示期間を変更
- **ウィンドウサイズ**: 小(1×1)、横長(2×1)、大(2×2)から選択
- **ドラッグ&ドロップ**: リサイズハンドルで直感的にサイズ変更

### 5. 設定画面

アプリの設定を変更する画面です。

- **目標設定**:
  - 目標タイプ: 減量・増量・維持
  - 目標体重
  - 1日の目標カロリー・PFC
- **1RM計算式**: Epley式・Brzycki式・Lombardi式・O'Conner式から選択

## 使い方のコツ

### 毎日の記録ルーティン

1. **朝**: 起床後に体重を測定・記録
2. **食事ごと**: 食べたものをすぐに記録（後からだと忘れがち）
3. **トレーニング時**: ジムでリアルタイムにセットを記録
4. **夜**: 分析画面で1日を振り返り

### 効果的なデータ活用

- **体重**: 毎日同じ時間（朝起きてトイレ後がおすすめ）に計測
- **カロリー**: 目標に対して±10%以内を目指す
- **トレーニング**: ボリューム（重量×回数×セット数）の増加を確認
- **分析**: 週単位で体重とカロリーの相関を確認

## オフライン対応

このアプリはPWA（Progressive Web App）として設計されており、インターネット接続がなくても動作します。

- データはすべてデバイス内に保存
- 一度アクセスすればオフラインでも使用可能
- ホーム画面に追加してネイティブアプリのように使用可能

## データのバックアップ

現在のバージョンでは、データはブラウザのローカルストレージ（IndexedDB）に保存されています。

**注意**: ブラウザのデータを削除すると記録も消えます。定期的なバックアップ機能は今後のバージョンで追加予定です。
